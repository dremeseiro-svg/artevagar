Manual de Deriva Persoal / Sociedad Errante
Por Daniel Remeseiro
Proxecto editorial e artístico de procesos antifráxiles.
Este repositorio contén o código fonte do Manual de Deriva Persoal, un "book" interactivo deseñado para documentar a colisión entre a intención do artista e a vontade do azar.
O Concepto
Baseado na filosofía de Antifraxilidade e na práctica da Vía Negativa, este traballo utiliza a experimentación con pranchas de xelatina para transformar o erro nun catalizador creativo. O obxectivo é rescatar a Sinal dentro do Caos, permitindo que a obra se fortalece coas perturbacións do proceso.
Características Técnicas
•	Formato Dual: Deseñado para ser unha galería interactiva en web e un dossier profesional en formato PDF (A4) mediante impresión optimizada.
•	Interatividade: O usuario pode interactuar coas imaxes para revelar o rastro de cor oculto tras o caos do branco e negro.
•	Deseño Minimalista: Tipografías Cormorant Garamond e Montserrat para unha estética editorial de alta gama.
